# ファイル仕分君

ファイル仕分君は、ファイルのメタデータ（画像のEXIF情報、ファイルシステム属性など）に基づいて、
ファイルを自動的に整理するデスクトップアプリケーションです。

## ローカルでのビルド

### 前提条件

-   Python 3.9以上

### セットアップ

```bash
pip install -r requirements.txt
```

### 実行可能ファイルのビルド

```bash
pyinstaller main.spec
```

## 欲しい機能
- 上書きせず名前変更して移動する
- アイコン変更する
- 設定方法わかりやすくする
- mypy導入する
